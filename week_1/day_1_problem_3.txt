Problem 3 - Best Time to Buy and Sell Stock

Example:
Input: prices = [7,1,5,3,6,4]
Output: 5

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float('inf')
        max_profit = 0
        
        for price in prices:
            if price < min_price:
                min_price = price
            elif price - min_price > max_profit:
                max_profit = price - min_price
        
        return max_profit

Time Complexity: O(n)
Space Complexity: O(1)
