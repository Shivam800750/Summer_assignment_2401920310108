LeetCode Problem: Maximum Subarray (Problem #53)
Submission ID: 2025160501

Python Solution:

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        max_sum = nums[0]
        current_sum = nums[0]
        for num in nums[1:]:
            current_sum = max(num, current_sum + num)
            max_sum = max(max_sum, current_sum)
        return max_sum

Explanation:
- Dynamic programming approach to find the maximum sum of a contiguous subarray
- We maintain two variables:
  1. max_sum: The maximum sum found so far
  2. current_sum: The maximum sum ending at the current position
- At each step, we decide whether to add the current number to the existing sum or start a new subarray
- Time Complexity: O(n)
- Space Complexity: O(1)
