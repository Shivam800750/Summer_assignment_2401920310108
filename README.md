This repository contains my daily and weekly submissions for the 4-Week DSA & OOPS Summer Assignment. The objective of this program is to build consistency, improve algorithmic problem-solving skills, and practice writing clean, optimal code.

Each solution in this repository is written with the assignment's core evaluation criteria in mind:
*   **Optimal Problem-Solving:** Choosing the right data structures and algorithms.
*   **Complexity Analysis:** Clearly defining the Time and Space Complexity for every solution.
*   **Clean Code:** Adhering to proper coding practices, meaningful variable naming, and readability.
*   **Consistency:** Pushing code daily without building up pending work.
